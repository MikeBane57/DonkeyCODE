Welcome to DonkeyCode

A few quick steps and you’re ready to use user scripts and saved sessions.

  1. 1. Allow website access

    DonkeyCode needs permission to run on http and https pages so your scripts can inject. We don’t use broad “read all data” at install—you choose here.

  2. 2. Pin to toolbar (optional)

    Click the puzzle piece (🧩) in the browser toolbar → find DonkeyCode → click the pin so the donkey icon stays visible for quick access.

  3. 3. Get started

    This closes the welcome tab, opens DonkeyCode from the toolbar, and loads your script list automatically.
